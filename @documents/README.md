# مستندات پروژه android_app

## ۱. معرفی پروژه

### هدف
پروژه **android_app** یک اپلیکیشن موبایل فروشگاهی است که با هدف ارائه تجربه خرید آنلاین به کاربران فارسی‌زبان طراحی شده است. این اپلیکیشن با تمرکز بر رابط کاربری مدرن، سرعت بالا و سهولت استفاده توسعه یافته است.

### زبان برنامه‌نویسی و تکنولوژی‌ها
- **زبان برنامه‌نویسی:** Dart
- **فریم‌ورک:** Flutter (قابل اجرا روی Android و iOS)
- **کتابخانه‌ها:**
  - http (ارتباط با API)
  - google_fonts (مدیریت فونت‌ها)
  - carousel_slider (اسلایدر تصاویر)
  - flutter_localizations (پشتیبانی از زبان فارسی)
- **فونت:** Vazirmatn (فونت فارسی)

### مخاطبان
- کاربران موبایل (Android و iOS)
- توسعه‌دهندگان علاقه‌مند به توسعه اپلیکیشن‌های فروشگاهی

---

## ۲. نیازمندی‌ها

### قابلیت‌های اصلی
- رابط کاربری کاملاً فارسی و راست‌چین
- نمایش اسلایدر بنر و دسته‌بندی‌ها
- نمایش محصولات ویژه، پرفروش و جدید
- مشاهده و جستجوی دسته‌بندی‌ها و محصولات
- مشاهده جزئیات محصول و ثبت نظر کاربران
- ارتباط با API فروشگاه (WooCommerce)

### الزامات غیرعملکردی
- **امنیت:** استفاده از HTTPS برای ارتباط با API
- **کارایی:** بهینه‌سازی برای اجرا روی دستگاه‌های مختلف
- **قابلیت نگهداری:** ساختار ماژولار و مستندسازی مناسب
- **پشتیبانی از فونت فارسی**

### وابستگی‌ها
- Flutter SDK (حداقل نسخه 3.8.0)
- Android SDK و iOS SDK
- NDK نسخه 29.0.13113456
- کتابخانه‌های ذکر شده در pubspec.yaml

## ۳. ساختار پروژه

### معماری
پروژه از معماری استاندارد Flutter پیروی می‌کند و شامل ویجت‌های Stateless و Stateful برای مدیریت صفحات و اجزای مختلف است. ارتباط با API به صورت مستقیم و با استفاده از کتابخانه http انجام می‌شود.

### ساختار فایل‌ها/پوشه‌ها
```
seify_app/
├── android/          # کدهای مربوط به Android
├── ios/              # کدهای مربوط به iOS
├── lib/              # کدهای اصلی برنامه (main.dart)
├── assets/           # منابع برنامه (فونت‌ها و تصاویر)
├── test/             # تست‌های واحد
└── pubspec.yaml      # فایل تنظیمات پروژه
```

## ۴. توضیح کدها

### شرح ماژول‌ها و توابع کلیدی

#### main.dart
- **MyApp:** نقطه ورود برنامه و تنظیمات تم و زبان.
- **CustomHeader:** هدر صفحه اصلی شامل لوگو و جستجو.
- **HomePage:** صفحه اصلی شامل اسلایدر، دسته‌بندی‌ها و محصولات.
- **BannerSlider:** اسلایدر بنر با قابلیت اسلاید خودکار.
- **CategoryMenuSlider:** نمایش دسته‌بندی‌ها به صورت افقی.
- **FeaturedProducts, BestSellingProducts, NewProducts:** نمایش محصولات ویژه، پرفروش و جدید.
- **CategoriesPage:** دریافت و نمایش دسته‌بندی‌ها از API.
- **ProductsPage:** دریافت و نمایش محصولات یک دسته‌بندی خاص.
- **ProductDetailPage:** نمایش جزئیات محصول، تصاویر، توضیحات و ثبت/نمایش نظرات کاربران.

#### مثال کد: دریافت دسته‌بندی‌ها از API
```dart
Future<void> fetchCategories() async {
  final url = Uri.parse('https://seify.ir/wp-json/wc/v3/products/categories?...');
  final response = await http.get(url);
  if (response.statusCode == 200) {
    setState(() {
      categories = json.decode(response.body);
      isLoading = false;
    });
  }
}
```

#### ورودی/خروجی توابع
- ورودی: اطلاعات از API (JSON)
- خروجی: نمایش داده‌ها در ویجت‌های مختلف

## ۶. راهنمای استفاده

### اجرای برنامه
- پس از اجرای `flutter run`، اپلیکیشن روی شبیه‌ساز یا دستگاه واقعی اجرا می‌شود.
- صفحه اصلی شامل اسلایدر، دسته‌بندی‌ها و محصولات است.
- با انتخاب هر دسته‌بندی یا محصول، جزئیات مربوطه نمایش داده می‌شود.
- کاربران می‌توانند نظرات خود را برای هر محصول ثبت کنند.

#### مثال کاربردی: مشاهده محصولات یک دسته‌بندی
1. ورود به صفحه دسته‌بندی‌ها
2. انتخاب یک دسته
3. مشاهده لیست محصولات و انتخاب هر محصول برای مشاهده جزئیات
