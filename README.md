# android_app

A Flutter mobile application project.

## Version Information
- Current Version: 1.1.0+1
- Last Updated: 2024

## Features
- Modern UI with Material Design
- RTL support for Persian language
- Product listing and categories
- Product details with image gallery
- User reviews system
- Responsive layout
